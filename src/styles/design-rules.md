# Design Rules

## Spacing

Use only the following spacing utilities:

2 → 0.5rem  
4 → 1rem  
6 → 1.5rem  
8 → 2rem  
12 → 3rem  
16 → 4rem  
24 → 6rem

Avoid values like mt-10, px-7, etc.

## Border radius

- Cards: rounded-lg
- Buttons: rounded-md

## Borders

- Default: border border-border
- Subtle separators: border-white/10
